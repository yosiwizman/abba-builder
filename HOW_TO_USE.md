## 🔧 Troubleshooting

### App Won't Start

1. Check Node.js is installed: `node --version`
2. Run `npm install` to ensure dependencies
3. Use `Abba-AI-Builder-Stable.bat` for auto-restart

### Build Fails

- AI will automatically retry with different template
- Check console for specific error messages
- Knowledge base has solutions for common errors

### Performance Issues

- Close unnecessary browser tabs
- Ensure 8GB+ RAM available
- Use local projects for faster builds
